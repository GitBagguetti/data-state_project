# Final Project

Brief overview of the notebooks in this folder:

## `0_exploration.ipynb`

Preliminary exploration of the dataset for the project proposal.

## `1_networks.ipynb`

Copy of the `weapons.ipynb` notebook provided in the course materials, with two main modifications:

- Added exploration of weapons flows by year to identify suitable historical focal points for the analysis.
- Tweaked the network-building code to focus on the two periods chosen.

The final structural analysis of the networks will follow, as will the addition of tie properties.

## `2_edge_quality.ipynb`

Extraction and construction of tie property characteristics (aid and reasons given for arms transfers). Topic modeling with BERTopic has a pipeline set up but requires a few steps of final refinement to work for this project’s purposes.
